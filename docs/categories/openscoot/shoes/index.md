# Drum brake shoes

One page per drum-brake shoe or shoe-bearing assembly, generated from `data/openscoot/shoes/*.yml`. Both stock/OEM and generic / aftermarket entries can appear here; the `OEM` column on each scooter page tells them apart.

This slice covers drum-brake friction parts only. Disc-brake pads live under [Brake pads](../pads/index.md), and brake cables, levers, front forks, and complete wheels are repair parts outside this class.

The current evidence is thin. For the Segway-Ninebot Max G30 family, the discoverable replacement is usually a complete front drum-brake assembly with shoes installed, not loose standalone shoe linings. The page still tracks it here because it is the brake-side wear path for drum-braked units, but the part rows keep that assembly caveat visible.

See [the data README](https://github.com/openconsumables/openconsumables.github.io/blob/master/data/README.md) for the schema and how to add or correct an entry.
