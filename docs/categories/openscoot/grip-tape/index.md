# Replacement grip tape

One page per pre-cut adhesive grip tape / foot mat, generated from `data/openscoot/grip-tape/*.yml`. Generic rolls of non-slip tape are not listed here: they can be trimmed to fit many scooters, but they are not a discoverable scooter-specific consumable.

Grip tape is the weakest OpenScoot wear class in this reference layer. Rebuy frequency is low, OEM spare-part signal is thin, and many sellers publish photos rather than measured drawings. The entries here answer the narrow compatibility question: whether a seller is claiming a pre-cut mat for a known deck profile.

A grip-tape entry lists the scooters it fits, with provenance per claim. Most claims are `marketplace-claim`, because replacement foot mats are usually white-label parts and seller compatibility lists copy each other.

See [the data README](https://github.com/openconsumables/openconsumables.github.io/blob/master/data/README.md) for the schema and how to add or correct an entry.
