# Brake pads

One page per brake pad, generated from `data/openscoot/pads/*.yml`. Both stock/OEM and generic / aftermarket pads can have entries; the `OEM` column on each scooter page tells them apart.

A pad entry lists the scooters it fits, with provenance per claim. The lowest trust tier (`marketplace-claim`) is published but clearly labelled, because aftermarket compatibility lists are usually not independent evidence.

This slice covers disc-brake pads only. Drum brake shoes and brake discs / rotors are separate part classes and are not folded into pads.

See [the data README](https://github.com/openconsumables/openconsumables.github.io/blob/master/data/README.md) for the schema and how to add or correct an entry.
