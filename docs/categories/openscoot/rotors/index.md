# Replacement rotors

One page per brake disc / rotor, generated from `data/openscoot/rotors/*.yml`. Both stock/OEM and generic / aftermarket rotors can have entries; the `OEM` column on each scooter page tells them apart.

A rotor entry lists the scooters it fits, with provenance per claim. The lowest trust tier (`marketplace-claim`) is published but clearly labelled, because aftermarket compatibility lists are usually not independent evidence.

This slice covers disc-brake rotors only. Brake pads, drum brake shoes, and brake calipers are separate part classes and are not folded into rotors.

See [the data README](https://github.com/openconsumables/openconsumables.github.io/blob/master/data/README.md) for the schema and how to add or correct an entry.
