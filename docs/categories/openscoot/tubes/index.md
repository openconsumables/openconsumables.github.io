# Inner tubes

One page per inner tube, generated from `data/openscoot/tubes/*.yml`. Tube entries use the same axle / wheel-mount slots as tires because a tube is sized to the tire and rim it sits inside.

This slice covers tubes for tubed pneumatic tires only. Tubeless self-healing tires, tubeless valves, sealant, patch kits, and solid / honeycomb tires are separate cases. A solid tire takes no tube by definition.

The Segway-Ninebot Max G30 is intentionally absent from the current tube table. Its stock tire is a 10-inch self-healing tubeless pneumatic, so a tube only becomes relevant after an aftermarket tubed-tire conversion. That conversion path is real but not the stock compatibility profile captured here.

See [the data README](https://github.com/openconsumables/openconsumables.github.io/blob/master/data/README.md) for the schema and how to add or correct an entry.
