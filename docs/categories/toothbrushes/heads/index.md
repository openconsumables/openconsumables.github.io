# Toothbrush heads

One page per head, generated from `data/heads/*.yml`. Both OEM and generic / clone heads have entries; the `OEM` column on each handle page tells them apart.

A head entry lists the handles it fits, with provenance per claim. The lowest trust tier (`marketplace-claim`) is published but clearly labelled, because marketplace compatibility lists are usually not independent evidence.

See [the data README](https://github.com/openconsumables/openconsumables.github.io/blob/main/data/README.md) for the schema and how to add or correct an entry.
