# Electric shavers

One page per shaver body, generated from `data/openshave/units/*.yml`. Each page lists the replacement heads that fit, with provenance for every claim.

The data layer is best-effort coverage. A unit with no measurements still has a page if at least one verifiable source supports its existence and at least one head claim or head-mount assignment. Measurement upgrades a row from `manufacturer-claim` to `measured`; it does not gate publishing.

See [the data README](https://github.com/openconsumables/openconsumables.github.io/blob/master/data/README.md) for the schema and how to add or correct an entry.
