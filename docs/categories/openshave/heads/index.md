# Replacement heads

One page per head, generated from `data/openshave/heads/*.yml`. Both OEM and generic / aftermarket heads have entries; the `OEM` column on each shaver page tells them apart.

A head entry lists the shavers it fits, with provenance per claim. The lowest trust tier (`marketplace-claim`) is published but clearly labelled, because aftermarket compatibility lists are usually not independent evidence. OpenShave evidence is heavier on marketplace-claim than OpenAir or OpenScoot; that is an honest reflection of the available material, not a target state.

See [the data README](https://github.com/openconsumables/openconsumables.github.io/blob/master/data/README.md) for the schema and how to add or correct an entry.
